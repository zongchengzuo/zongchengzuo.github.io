前言

Posted on 2017-09-13 | In Python计算机视觉手册 | Visitors:

今天，图像和视频无处不在。在线照片分享网站和社交网络上的图像已打数十亿之多。当前主流搜索引擎都已经支持图像搜索。几乎所有手机都有内置的摄像头，吃饭之前拍照片去景点拍照片等都是很正常的生活方式。

计算机视觉研究的就是设计算法来理解图像中有什么，对象之间的关系是怎样的。计算机视觉应用非常广泛，例如有图像搜索、机器人导航、医学图像分析、照片管理等。

本人所学专业为遥感图像处理，加上现在从事的工作也是与此相关，利用工作之余对此领域的心得体会进行总结，为自己走过的路树下一块路牌，多年以后回忆之时也算是有迹可循。编程语言用的是Python，Python拥有很多免费的的图像处理(PIL)、数学计算(NumPy)和数据挖掘模块(pandas)。

写作本专栏的时候，有以下几点申明。

因为本人水平所限，提出了很多问题但是自己也无法回答。 使用免费且开源的软件。
保持各个章节内容的完整性和独立性。 很多方向只是点到为止，既是因为水平所限难以展开，也是计算机视觉领域无比辽阔，实在是无法详尽。

总之，如果你对计算机视觉感兴趣，希望本专栏能给你带来收获。

###本系列主要的内容

·用 Python 对图像进行实战编程。
·现实世界中各种应用背后的计算机视觉技术的理解。
·一些基本算法，以及如何实现并应用这些算法。

所包含的代码示例包含物体识别、基于内容的图像检索、光学字符识别、光流跟踪、三维重建、立体成像、增强现实、姿态估计、全景创建、图像分割、降噪、图像分组等内容。

###章节概览
第一节 介绍处理图像的基本工具及本书用到的核心 Python 模块，同时涵盖了很多贯穿全章节的基础例子
第二节 检测图像兴趣点的方法，以及怎样使用它们在图像间寻找对应点和对应区域
第三节 描述图像间基本的变换及计算方法，从图像扭曲到创建全景图像
第四节 如何对相机建模，怎样使用投影从三维空间变换到二维图像，并估计相机视点
第五节 如何对具有相同场景、多视图几何的图像进行处理，以及怎样从二维图像计算三维重建
第六节 介绍一些聚类方法，并展示如何基于内容相似性对图像进行分组
第七节 如何建立有效的图像检索技术，以便能够存储图像的表示，并基于图像的视觉内容搜索图像
第八节 描述图像分类算法，以及怎样使用它们识别图像中的物体
第九节 图像的语义分割
第十节 怎样使用 OpenCV 的 Python 接口，以及如何处理视频及摄像头的输入

###计算机视觉简介
计算机视觉是一门对图像中的信息进行自动提取的学科。信息的内容相当广泛，包括三维模型、相机位置、目标检测与识别，以及图像内容的分组与搜索等。

计算机视觉有时试图模拟人类视觉，有时使用大数据和统计方法，而有时又使用几何方法。

可以说计算机视觉混合了编程、建模和数学理论，实在是很难全面掌握。我本着“力求简单，又不影响理解”的精神，有意用最少的理论来展示这些内容。
